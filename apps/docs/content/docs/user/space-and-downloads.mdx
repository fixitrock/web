## Open Space

Start from `/space`.

This is the main file and download surface for repair users. It is connected to the product's branded content areas, including:

- Firmware
- Apps
- Games
- USB Drivers
- Flash Tools
- iCloud

## How to Browse

1. Open `/space`.
2. Use the search layer or route navigation to jump into a folder or file group.
3. Open a folder page such as `/space/Drivers` or `/space/Flash-Tool`.
4. Read any available README or preview notes before downloading.

## What Users Should Expect

- Category-style navigation for repair resources
- Search commands that help jump between file areas
- Branded preview and metadata pages for Space content
- Shareable routes for specific folders or pages

## Tips for Faster File Discovery

- Search by brand name first, then model.
- Use the specialist routes when you already know the resource type.
- Prefer the exact folder route when sharing with teammates because it reduces search ambiguity.
