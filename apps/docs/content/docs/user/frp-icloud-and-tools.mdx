Fix iT Rock exposes specialist repair areas directly through dedicated routes.

## Main Specialist Sections

| Route | Purpose |
| --- | --- |
| `/frp` | Factory Reset Protection content and related unlock resources |
| `/space/iCloud` | Apple and iCloud-related material |
| `/space/Drivers` | Driver packs for device connection and service work |
| `/space/Flash-Tool` | Flashing and unlock tool bundles |

## When to Use Each Area

- Use `/frp` when the device is blocked by reset protection and you need the FRP workflow.
- Use `/space/iCloud` when working on Apple-specific recovery or lock scenarios.
- Use `/space/Drivers` before connecting a device that needs vendor or chipset drivers.
- Use `/space/Flash-Tool` when the next step is flashing, unlocking, or low-level service work.

## Safe Usage Notes

- Confirm the exact brand and model before downloading any file.
- Match drivers and flashing tools to the chipset or OEM family where possible.
- Keep a clean local folder structure when saving downloads from multiple tool areas.
