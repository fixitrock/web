## Your Store Address

Every seller storefront is anchored to a username route:

`/@username`

This URL is the base for both your public profile and your private workspace.

## Public Profile

The seller home route usually contains:

- Seller name and avatar
- Public bio or shop identity
- Public tabs such as products or activity
- Links into the seller's catalog

## Workspace Shortcuts

For seller roles, the app can expose installable shortcut targets such as:

- Products
- POS
- Settings
- Brands
- Categories

The exact shortcut set depends on the seller role. Higher-role sellers can access more catalog management tools.

## First Setup Checklist

1. Finish login and choose the correct permanent username.
2. Open `/@username/settings`.
3. Upload a clear avatar and store imagery.
4. Write a short shop description customers can understand.
5. Confirm your public storefront looks correct on `/@username`.

## Role-Aware Access

The project checks:

- Whether the current session belongs to the same username
- Whether the session role matches the required seller role
- Whether the requested seller page is private

If any of those checks fail, the user is sent back to the public storefront view.
