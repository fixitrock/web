Fix iT Rock combines two big jobs in one product:

1. Give repair users fast access to Space downloads, drivers, FRP files, iCloud resources, and tools.
2. Give sellers a live storefront with products, categories, brands, POS, orders, stock, and settings.

This docs app is written around the routes and features that already exist in the project.

## Main Product Areas

| Area | Route | What it is for |
| --- | --- | --- |
| Space | `/space` | Firmware, apps, games, drivers, flashing tools, and repair files |
| FRP | `/frp` | Factory Reset Protection help and related unlock resources |
| iCloud | `/space/iCloud` | Apple and iCloud-specific content |
| Drivers | `/space/Drivers` | USB driver downloads |
| Flash Tools | `/space/Flash-Tool` | Flashing and unlock utilities |
| Seller storefront | `/@username` | Public store page with tabs and profile |
| Seller workspace | `/@username/products`, `/pos`, `/orders`, `/settings` | Private operating area for the seller |
