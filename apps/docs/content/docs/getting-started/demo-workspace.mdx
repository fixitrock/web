## Recommended Demo Story

1. Log in with the seller staging account.
2. Open `/@rockmobile/products` and show the sample catalog.
3. Use `/@rockmobile/pos` to create a quick order.
4. Open the generated order or receipt view.
5. Switch to the customer perspective and show the public storefront or receipt link.

## Keep It Clean

- Never reuse demo OTP values in production.
